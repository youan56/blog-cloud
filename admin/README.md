# 星绘 sama 博客管理后台

基于 Vercel + CloudBase 的博客管理后台

## 技术栈

- React 18 + TypeScript
- Vite
- React Router
- CloudBase SDK

## CloudBase 配置

- 环境ID: `a-d6gb40txde961926e`
- 区域: `ap-shanghai`
- 认证方式: 用户名密码

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

项目将部署到 Vercel，连接 CloudBase 数据库管理博客内容。
